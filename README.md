# 📊CountMe Learning Analytics

## Student Performance Analysis Using Python

### Overview

CountMe Learning Analytics is a data analytics project developed to evaluate students' academic performance in a mathematics tutoring program. Using Python, the project analyzes score progression, attendance, and homework completion to generate meaningful educational insights that support data-driven decision-making.

---

## Project Objectives

- Analyze students' learning progress.
- Compare Pre-Test, Mid-Test, and Post-Test scores.
- Measure learning improvement.
- Evaluate attendance and homework completion.
- Identify students who need additional academic support.

---

## Tools

- Python
- Pandas
- Matplotlib
- Microsoft Excel
- Google Colab

---

## Dataset

This project uses a simulated dataset consisting of **30 senior high school students (Grade 10–12)**.

The dataset includes:

- Student Information
- Academic Scores
- Attendance
- Homework Completion
- Learning Outcomes
